# Initial Understanding

The paper focuses on reducing localization errors in
UWB TDoA indoor positioning 
The proposed solution uses an unsupervised machine learning approach 
based on Deep Embedded Clustering (DEC).

The main goal is to mitigate errors caused by challenging propagation 
conditions and improve localization accuracy without requiring labeled datasets.
